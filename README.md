# Jingan-Bot
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/miqbal-22/mybot
> cd mybot
>bash instal.sh
> npm install
```
###### Run
```bash
> node index.js
```
